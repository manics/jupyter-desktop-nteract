# Jupyter Desktop Nteract

[![Build](https://github.com/manics/jupyter-desktop-nteract/actions/workflows/build.yaml/badge.svg)](https://github.com/manics/jupyter-desktop-nteract/actions/workflows/build.yaml)
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/manics/jupyter-desktop-nteract/HEAD?urlpath=desktop)

A container image providing [nteract](https://www.nteract.io/) running on [Jupyter Desktop MATE](https://github.com/manics/jupyter-desktop-mate/).

## Usage

```sh
docker run -p127.0.0.1:8888:8888 ghcr.io/manics/jupyter-desktop-nteract:latest
```

Open the URL shown, and click on the Desktop launcher.
